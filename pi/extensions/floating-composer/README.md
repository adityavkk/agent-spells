# floating-composer

Installable alternative to footer-only styling.

Features:
- opencode-inspired unified composer card
- replaces editor component with a darker bordered floating panel
- integrates footer/status inside same visual surface
- profile-aware model resolution for synthetic `profiles/<profile:role>` models
- quota bars follow the resolved concrete provider

Usage:
- enable `floating-composer`
- disable `minimal-footer` and `floating-footer`

Path:
- `~/dev/agent-spells/pi/extensions/floating-composer/`
